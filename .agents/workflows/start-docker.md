---
description: Iniciar o sistema Monitor-Flow usando Docker
---
Para iniciar ou reiniciar o sistema Monitor-Flow usando os containers Docker, siga estes passos:

// turbo-all
1. Execute o comando `docker compose up --build -d` no repositório `c:\Projetos\monitor-flow` para reconstruir suas imagens se necessário, e subir os containers em background.
2. Execute o comando `docker ps` para diagnosticar e visualizar os serviços ativos e as portas mapeadas na máquina local.
